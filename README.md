# photo_album_manager


This is the plug-in can quickly get album resources, support for android and iOS

这是可以快速获取相册资源的插件，支持安卓和iOS



## install


```yaml
dependencies:
album_manager: 0.0.1
```

## import

```dart
import 'package:album_manager/photo_album_manager.dart';
```
